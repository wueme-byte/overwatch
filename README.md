# Overwatch

Track Telegram Gift prices across **Fragment** and **GetGems** — in one place, always sorted by best price.

🔗 **[overwatchgifts.help](https://overwatchgifts.help)**  
🤖 **[@Overwatch_Gifts_bot](https://t.me/Overwatch_Gifts_bot)**

---

## What it does

- Aggregates listings from Fragment and GetGems in real time
- Filters by model, price range, and attributes
- Shows the best available price across both marketplaces
- Highlights auction listings from Fragment
- Works as a Telegram Mini App — no install needed

---

## How to use

1. Open [@Overwatch_Gifts_bot](https://t.me/Overwatch_Gifts_bot) in Telegram
2. Tap **Start Overwatch**
3. Browse collections, filter by model, find the best price
